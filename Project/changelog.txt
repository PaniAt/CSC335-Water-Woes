Version 1.0.0 (25/06/2026)
- Implemented initial Pipe class
    - Pipe flowing to connected pipes
    - Pipe rotation
    - Pipe connections
    - Pipe types
- Added the basic graphical textures, currently unused

Version 1.1.0 (29/06/2026)
- Implemented initial PipeNetwork class
    - PipeNetworks' grid of pipes
    - PipeNetworks can add/set pipes
    - PipeNetworks can remove pipes
    - Alias for quickly constructing pipes
- The (currently unused) infrastructure for screen sizing, and other various
  window utilities which will be utilised later in the project
- Some stylisation of code documentation and layout

Version 1.2.0 (30/06/2026)
- Implemented PipeNetwork flowing
    - PipeNetworks can flow water into pipes
    - PipeNetworks can automatically update pipe connections
    - PipeNetworks can simulate a full step in the flow model
- Added Pipe::connectsTo() to check for connections
- Added Pipe::disconnect() to disconnect pipes, instead of setting to null
- Replaced binary notation with hex notation for Pipe connect transforms
- Tiny bit more stylisation

Version 1.3.0 (01/07/2026)
- Implemented visualisations for PipeNetworks
    - PipeNetworks can create a window using JFrame
    - PipeNetworks visualise the grid and all pipes in the tilemap
    - PipeNetworks position pipes correctly for display
    - PipeNetworks rotate pipes correctly for display (evil matrix math)
- Moved around a bit more code in Pipe::flow()
- Added a static main() method for VScode to run the program

Version 1.4.0 (03/07/2026)
- Implemented a basic model editor for PipeNetworks
    - PipeNetworks can get keyboard and mouse input from the user
    - PipeNetworks can add pipes via the editor
    - PipeNetworks can remove pipes via the editor
    - PipeNetworks can drain pipes via the editor
    - Added (currently unused) method for draining all pipes in a PipeNetwork
    - PipeNetwork editor can rotate pipes, although it is not very usable
- Added Pipe::getDirection() and Pipe::setDirection() for pipes

Version 1.5.0 (09/06/2026)
- Implemented a basic GUI and HUD for PipeNetworks
    - PipeNetworks display a MenuBar
    - PipeNetworks have MenuBar accelerators
    - PipeNetworks have a toggleable pipe selection overlay
    - Selection overlay has some general information about the program
- Added new method to count all non-null pipes in a PipeNetwork
- Moved parts of the screen rendering into separate functions
